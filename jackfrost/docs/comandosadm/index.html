Comandos de Administrador

Esses comandos podem ser executados apenas por administradores, e servem para regular o uso do bot em canais.

Habilitar comandos

Comandos são habilitados por padrão.
d$desabilitarcomandos para desabilitar.
d$habilitarcomandos para habilitar novamente.

Mensagens de boas vindas

Mensagens de boas vindas são mensagens que são enviadas sempre que um usuário entra no servidor.

Habilitar e desabilitar mensagens

As mensagens são desabilitadas por padrão.
d$desabilitarboasvindas para desabilitar
d$habilitarboasvindas para habilitar

Editar mensagem de boas vindas

Use o comando d$mensagemdeboasvindas para editar a mensagem.

Interessante!

No texto, use "{{user}}" para mencionar o novo membro do servidor.

Mensagens de Level Up

Mensagens de boas vindas são mensagens que são enviadas sempre que um usuário sobre de nível no servidor.

Habilitar e desabilitar mensagens

As mensagens são desabilitadas por padrão.
d$desabilitarlvup para desabilitar
d$habilitarlvup para habilitar

Editar mensagem de boas vindas

Use o comando d$mensagemdelvup para editar a mensagem.

Interessante!

No texto, use "{{user}}" para mencionar o novo membro do servidor.

Aumentar e diminuir XP

É possível editar a quantidade de XP de um usuário no servidor.

d$increasexp <quantidade> <usuário> para aumentar.
d$decreasexp <quantidade> <usuário> Para diminuir.
