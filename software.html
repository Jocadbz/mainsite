This is a fairly decent list of the software I use daily. Some stuff may be subject to change since I will forget about this page often...

programming

-> vim - Probably what I use the most. I use it with no modifications, it's quick, fast and pretty good. -> xed - Used when I need to edit massive codebases and replace a lot of stuff, and when vim isn't fast enough (rarely). -> arisu - Anything AI-related I do, I do through this. Fast, quick, smart and very efficient. Also, nothing better than to use your own software.

system

-> arch - It's my distro of choice since last year. Easy to package to, fast and gets out of my way. -> i3m - The best WM in the market btw. All others either try to complicate stuff or just suck. I use it as my main WM since 2022. -> alacritty - Works. -> nemo - Imagine a file browser that's fast, efficient, has every internet protocol supported natively and has customizable actions and shortcuts? Yeah, that's nemo. That's what every single DE out there should be using/copying. -> less - The best file pager of all time.
